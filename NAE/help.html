BlackLynx HPC

BlackLynx specializes in rapid search of big data sets without the need to transform or index the data. Our proprietary algorithms make the best use of available hardware (CPU, GPU or FPGA) to search raw unstructured text, CSV, JSON, XML and/or PCAP (Packet Capture Data) files natively. This document provides instructions for accessing your BlackLynx session in the Nimbix Cloud.
